==================================================================
  1. 简介
==================================================================
  本sdk主要封装了 OAuth1.0 和 OAuth2.0 的授权方法和20多个常用的
API请求的方法。
	
  若需了解更多API信息，请参考
     http://wiki.open.t.qq.com/index.php/API文档


==================================================================
  2. SDK压缩包内的重要资源介绍
==================================================================
  /src/main/java     sdk的项目源代码，
                     示例程序位于（com.tencent.weibo.example 包中）

  /src/doc           sdk中提供的方法的 java doc 文档

  /lib               sdk依赖的第三方库

  target/Java_SDK_v???.jar  本sdk编译得到的Jar库

  
==================================================================
  3. SDK基本使用方法
==================================================================
  先将SDK压缩包中的lib目录与target目录下的JAR包，添加到您项目的
编译路径；

  然后参考API文档和示例程序（位于com.tencent.weibo.example）理解
SDK的调用方法。
